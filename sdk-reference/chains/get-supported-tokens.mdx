---
title: 'Get Supported Tokens for Chain'
description: 'Retrieve all supported tokens available on a specified blockchain network'
---

Retrieves all supported tokens available on a specified blockchain network.

<Tabs>
<Tab title="Try it">


```typescript
import { crapi, Chainrails } from "@chainrails/sdk"

Chainrails.config({
  api_key: "your_api_key_here"
})

const tokens = await crapi.chains.getSupported({
  chain: "BASE_MAINNET"
})

console.log("Supported tokens:", tokens)
```
